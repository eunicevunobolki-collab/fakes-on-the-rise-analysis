# fakes-on-the-rise-analysis
This project analyzes trends in counterfeit goods using data analysis and visualization in R and Quarto.  Cleaned and processed real-world data using R Conducted exploratory data analysis to identify key trends and patterns Built visualizations to communicate findings clearly Presented results in a structured Quarto report
This project demonstrates my ability to work with data, generate insights, and present results in a clear, user-friendly format.
